# AGENTS.md — OMC + Superpowers 프로젝트 런북

## 프로젝트 개요

- 이 프로젝트는 **OMC + Superpowers 프로젝트 런북**이다.
- 학생이 원하는 프로젝트 주제를 입력하면 Day 7·Day 8 진행 순서를 만든다.
- 수업 목적은 많은 기능이 아니라 **좋은 작업 절차를 눈에 보이게 만드는 것**이다.

---

## 기술 규칙

- 프레임워크는 강제하지 않는다.
- 브라우저 `localStorage`로만 저장한다.
- 화면은 복잡한 칸반 보드가 아니라 **단계형 런북 GUI**로 만든다.
- 텍스트가 모바일에서 겹치지 않게 한다.

---

## 하네스 규칙

- 불명확하면 **Superpowers `brainstorming`** 으로 질문한다.
- 여러 관점이 필요하면 **OMC `team`** 으로 기획자·구현자·검토자 의견을 나눈다.
- 구현 전에는 **`PLAN.md`를 만든다**.
- 완료 전에는 **`VERIFY.md`에 실제 확인 결과를 남긴다**.
- 확인하지 못한 것은 성공한 것처럼 쓰지 않는다.

---

## 금지

- API 키나 토큰을 파일에 하드코딩하지 않는다.
- Day 7·Day 8 최종 프로젝트를 지금 완성하려고 하지 않는다.
- 불필요한 라이브러리를 추가하지 않는다.
